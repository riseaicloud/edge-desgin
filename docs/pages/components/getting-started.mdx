# 组件库快速开始

## 安装

```bash
pnpm install @riseaicloud/ui
```

## 使用

### 1. 导入样式

在应用入口导入 CSS：

```tsx
import '@riseaicloud/ui/styles'
```

### 2. 使用组件

```tsx
import { Button } from '@riseaicloud/ui'

export function App() {
  return <Button>Click me</Button>
}
```

### 3. 导入工具函数

```tsx
import { cn } from '@riseaicloud/ui'

// 合并 Tailwind 类名
const className = cn('px-4 py-2', isActive && 'bg-blue-500')
```
