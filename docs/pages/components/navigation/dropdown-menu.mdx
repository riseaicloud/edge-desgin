# Dropdown Menu

A contextual menu that appears below a trigger, supporting items, checkboxes, radio buttons, and submenus, built on Radix UI DropdownMenu.

## Import

```tsx
import {
  DropdownMenu,
  DropdownMenuTrigger,
  DropdownMenuContent,
  DropdownMenuItem,
  DropdownMenuLabel,
  DropdownMenuSeparator,
  DropdownMenuCheckboxItem,
  DropdownMenuRadioGroup,
  DropdownMenuRadioItem,
  DropdownMenuShortcut,
  DropdownMenuSub,
  DropdownMenuSubTrigger,
  DropdownMenuSubContent,
} from '@riseaicloud/ui'
```

## With checkboxes

```tsx
import { useState } from 'react'

function ViewOptionsMenu() {
  const [showStatus, setShowStatus] = useState(true)
  return (
    <DropdownMenuContent>
      <DropdownMenuLabel>视图选项</DropdownMenuLabel>
      <DropdownMenuSeparator />
      <DropdownMenuCheckboxItem
        checked={showStatus}
        onCheckedChange={setShowStatus}
      >
        显示状态列
      </DropdownMenuCheckboxItem>
    </DropdownMenuContent>
  )
}
```

## API

### DropdownMenu (Root) Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `open` | `boolean` | — | Controlled open state |
| `defaultOpen` | `boolean` | `false` | Initial open state |
| `onOpenChange` | `(open: boolean) => void` | — | Callback when open state changes |
| `modal` | `boolean` | `true` | Whether to render in a modal context |

### DropdownMenuContent Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `align` | `'start' \| 'center' \| 'end'` | `'center'` | Alignment relative to trigger |
| `side` | `'top' \| 'right' \| 'bottom' \| 'left'` | `'bottom'` | Preferred side |
| `sideOffset` | `number` | `4` | Distance from trigger (px) |
| `className` | `string` | — | Additional CSS classes |

### DropdownMenuItem Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `inset` | `boolean` | `false` | Add left padding to align with items that have icons |
| `disabled` | `boolean` | `false` | Disabled state |
| `onSelect` | `(event: Event) => void` | — | Called when the item is selected |
| `className` | `string` | — | Additional CSS classes |
