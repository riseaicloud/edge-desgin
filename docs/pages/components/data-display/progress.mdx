# Progress

A horizontal progress bar indicating completion of a task or operation, built on Radix UI Progress.

## Import

```tsx
import { Progress } from '@riseaicloud/ui'
```

## Usage

<Preview center={false}>
  <Progress value={60} className="w-80" />
</Preview>

```tsx
<Progress value={60} />
```

## Values

<Preview center={false}>
  <div className="space-y-3 w-80">
    <Progress value={0} />
    <Progress value={25} />
    <Progress value={50} />
    <Progress value={75} />
    <Progress value={100} />
  </div>
</Preview>

```tsx
<Progress value={0} />
<Progress value={25} />
<Progress value={50} />
<Progress value={75} />

```

## Indeterminate

Omit `value` or pass `null` for indeterminate state (no indicator position):

```tsx
<Progress />
```

## API

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `number \| null` | — | Current progress (0–100). Omit for indeterminate |
| `max` | `number` | `100` | Maximum value |
| `className` | `string` | — | Additional CSS classes |

All Radix UI Progress Root props are supported.
