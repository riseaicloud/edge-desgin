# PropertyList

A grid of label-value pairs for displaying resource attributes and metadata. Commonly used inside `CollapsibleSection` to create collapsible property panels.

## Import

```tsx
import { PropertyList } from '@riseaicloud/ui'
```

## Usage

<Preview center={false}>
  <div className="w-full">
    <PropertyList
      items={[
        { label: "Name", value: "master-001" },

        { label: "Role", value: "Control Plane" },
        { label: "OS", value: "linux" },
        { label: "Architecture", value: "amd64" },
        { label: "Kernel", value: "5.15.0-138-generic" },
      ]}
    />
  </div>
</Preview>

## API

### PropertyList Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `items` | `PropertyItem[]` | required | Array of property items |
| `columns` | `1 \| 2 \| 3 \| 4` | `3` | Number of grid columns |
| `className` | `string` | — | CSS classes for the grid container |
| `labelClassName` | `string` | — | CSS classes for label elements |
| `valueClassName` | `string` | — | CSS classes for value elements |

### PropertyItem

| Field | Type | Default | Description |
|-------|------|---------|-------------|
| `label` | `string` | required | Property label (key) |
| `value` | `ReactNode` | required | Property value — string or custom node |
| `span` | `number` | `1` | Number of columns this item spans |

## Migration Guide

Replace the inline properties pattern used in 47+ layout files:

```tsx
// Before (repeated in every layout.tsx)
<div className="grid grid-cols-3 gap-x-12 gap-y-3 text-sm">
  <div className="flex items-center">
    <span className="text-gray-600 mr-2">Status:</span>
    <span className="text-gray-900">Running</span>
  </div>
  {/* ... repeat for each property */}
</div>

// After
<PropertyList
  items={[
    { label: "Status", value: <StatusIndicator variant="success" label="Running" /> },
    { label: "Cluster", value: "production (host)" },
    // ...
  ]}
/>
```
