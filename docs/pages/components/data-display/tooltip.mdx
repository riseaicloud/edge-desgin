# Tooltip

A popup that displays informative text when users hover or focus an element, built on Radix UI Tooltip.

## Import

```tsx
import { Tooltip, TooltipTrigger, TooltipContent, TooltipProvider } from '@riseaicloud/ui'
```

## Setup

Wrap your app (or page) with `TooltipProvider` once:

```tsx
// app/layout.tsx
import { TooltipProvider } from '@riseaicloud/ui'

export default function RootLayout({ children }) {
  return <TooltipProvider>{children}</TooltipProvider>
}
```

## API

### TooltipProvider Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `delayDuration` | `number` | `700` | Delay before tooltip opens (ms) |
| `skipDelayDuration` | `number` | `300` | Delay to re-open without waiting (ms) |

### Tooltip (Root) Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `open` | `boolean` | — | Controlled open state |
| `defaultOpen` | `boolean` | `false` | Initial open state |
| `onOpenChange` | `(open: boolean) => void` | — | Callback when open state changes |

### TooltipTrigger Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `asChild` | `boolean` | `false` | Render as child element |

### TooltipContent Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `side` | `'top' \| 'right' \| 'bottom' \| 'left'` | `'top'` | Preferred side |
| `sideOffset` | `number` | `4` | Distance from trigger (px) |
| `align` | `'start' \| 'center' \| 'end'` | `'center'` | Alignment relative to trigger |
| `className` | `string` | — | Additional CSS classes |
