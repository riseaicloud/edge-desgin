# Sheet (Drawer)

A panel that slides in from the edge of the screen, built on Radix UI Dialog.

## Import

```tsx
import {
  Sheet,
  SheetTrigger,
  SheetContent,
  SheetHeader,
  SheetTitle,
  SheetDescription,
  SheetFooter,
  SheetClose,
} from '@riseaicloud/ui'
```

## Usage

<Preview>
  <Sheet>
    <SheetTrigger asChild>
      <Button variant="outline">Open Panel</Button>
    </SheetTrigger>
    <SheetContent>
      <SheetHeader>
        <SheetTitle>Edit Configuration</SheetTitle>
        <SheetDescription>
          Adjust resource limits and environment variables.
        </SheetDescription>
      </SheetHeader>
      <div className="flex-1" />
      <SheetFooter>
        <SheetClose asChild>
          <Button variant="outline">Cancel</Button>
        </SheetClose>
        <SheetClose asChild>
          <Button>Save Changes</Button>
        </SheetClose>
      </SheetFooter>
    </SheetContent>
  </Sheet>
</Preview>

```tsx
<Sheet>
  <SheetTrigger asChild>
    <Button variant="outline">Open Panel</Button>
  </SheetTrigger>
  <SheetContent>
    <SheetHeader>
      <SheetTitle>Edit Configuration</SheetTitle>
      <SheetDescription>
        Adjust resource limits and environment variables.
      </SheetDescription>

      <SheetClose asChild>

```

## API

### Sheet (Root) Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `open` | `boolean` | — | Controlled open state |
| `defaultOpen` | `boolean` | `false` | Initial open state (uncontrolled) |
| `onOpenChange` | `(open: boolean) => void` | — | Callback when open state changes |

### SheetContent Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `side` | `'top' \| 'right' \| 'bottom' \| 'left'` | `'right'` | Direction the sheet slides in from |
| `showCloseButton` | `boolean` | `true` | Show the X close button |
| `className` | `string` | — | Additional CSS classes |

### SheetTrigger / SheetClose Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `asChild` | `boolean` | `false` | Render as child element |
