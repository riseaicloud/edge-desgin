# Dialog / Modal

A modal dialog that interrupts user flow for critical actions or focused tasks, built on Radix UI Dialog.

## Import

```tsx
import {
  Dialog,
  DialogTrigger,
  DialogContent,
  DialogHeader,
  DialogTitle,
  DialogDescription,
  DialogFooter,
  DialogClose,
} from '@riseaicloud/ui'
```

## Usage

确认删除对话框，用于节点组、用户、配置等不可逆操作。

<Preview bgColor="#EFF4F9">

    <DialogTrigger asChild>
      <Button variant="destructive">删除节点组</Button>
    </DialogTrigger>
    <DialogContent>
      <DialogHeader>
        <DialogTitle>确认删除</DialogTitle>
        <DialogDescription>
          删除后无法恢复，该操作将永久删除节点组及其关联配置。
        </DialogDescription>
      </DialogHeader>
      <DialogFooter>
        <DialogClose asChild>
          <Button variant="outline">取消</Button>

</Preview>

## 编辑对话框

编辑节点组、编辑配置字典等表单型弹窗。

```tsx
import { useState } from 'react'

function EditNodeGroupDialog() {
  const [open, setOpen] = useState(false)
  return (
    <Dialog open={open} onOpenChange={setOpen}>
      <DialogTrigger asChild>
        <Button variant="outline">编辑节点组</Button>
      </DialogTrigger>
      <DialogContent>
        <DialogHeader>
          <DialogTitle>编辑节点组</DialogTitle>
          <DialogDescription>
            修改节点组的名称和描述信息。
          </DialogDescription>
        </DialogHeader>
        {/* 表单内容 */}
        <DialogFooter>
          <DialogClose asChild>
            <Button variant="outline">取消</Button>
          </DialogClose>
          <Button onClick={() => setOpen(false)}>确定</Button>
        </DialogFooter>
      </DialogContent>
    </Dialog>
  )
}
```

## 创建对话框

创建节点组、创建 OTA 任务等新建类弹窗。

```tsx
<Dialog>
  <DialogTrigger asChild>
    <Button><Plus className="mr-2 h-4 w-4" /> 创建节点组</Button>
  </DialogTrigger>
  <DialogContent>
    <DialogHeader>
      <DialogTitle>创建节点组</DialogTitle>
    </DialogHeader>
    {/* 表单内容 */}
    <DialogFooter>
      <DialogClose asChild>
        <Button variant="outline">取消</Button>
      </DialogClose>
      <Button type="submit">确定</Button>
    </DialogFooter>
  </DialogContent>
</Dialog>
```

## API

### Dialog (Root) Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `open` | `boolean` | — | Controlled open state |
| `defaultOpen` | `boolean` | `false` | Initial open state (uncontrolled) |
| `onOpenChange` | `(open: boolean) => void` | — | Callback when open state changes |

### DialogContent Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | — | Additional CSS classes (default: centered modal, max-w-lg) |

A close button (`X`) is automatically rendered in the top-right corner.

### DialogTrigger Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `asChild` | `boolean` | `false` | Render as child element |

### DialogClose Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `asChild` | `boolean` | `false` | Render as child element |
