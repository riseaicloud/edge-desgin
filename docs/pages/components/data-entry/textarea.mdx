# Textarea

A multi-line text input field for longer content.

## Import

```tsx
import { Textarea } from '@riseaicloud/ui'
```

## Usage

<Preview center={false}>
  <Textarea placeholder="Enter a description..." className="w-80" />
</Preview>

```tsx
<Textarea placeholder="Enter a description..." />
```

<Preview center={false}>

    <Textarea id="description-preview" placeholder="Describe your resource..." />
  </div>
</Preview>

```tsx
import { Label } from '@riseaicloud/ui'

<div className="flex flex-col gap-1.5">
  <Label htmlFor="description">Description</Label>

```

## Controlled

```tsx
import { useState } from 'react'

function ControlledTextarea() {
  const [value, setValue] = useState('')
  return (
    <Textarea
      value={value}
      onChange={(e) => setValue(e.target.value)}
      placeholder="Type something..."
    />
  )
}
```

## Resize control

Use Tailwind classes to control resize behavior:

```tsx
<Textarea className="resize-none" placeholder="No resize" />
<Textarea className="resize-y" placeholder="Vertical resize only" />
```

## API

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `placeholder` | `string` | — | Placeholder text |
| `disabled` | `boolean` | `false` | Disabled state |
| `rows` | `number` | — | Number of visible text lines |
| `className` | `string` | — | Additional CSS classes |

All standard HTML `textarea` attributes are supported. Default `min-height` is `80px`.
