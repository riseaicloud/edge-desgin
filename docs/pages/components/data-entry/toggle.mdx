# Toggle / ToggleGroup

A two-state button that can be toggled on or off. `ToggleGroup` allows grouping multiple toggles with shared state.

## Import

```tsx
import { Toggle } from '@riseaicloud/ui'
import { ToggleGroup, ToggleGroupItem } from '@riseaicloud/ui'
```

## Toggle — Basic Usage

<Preview>

  </Toggle>
</Preview>

## API

### Toggle Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'default' \| 'outline'` | `'default'` | Visual style variant |
| `size` | `'default' \| 'sm' \| 'lg'` | `'default'` | Size of the toggle |
| `pressed` | `boolean` | — | Controlled pressed state |
| `defaultPressed` | `boolean` | `false` | Initial pressed state (uncontrolled) |
| `onPressedChange` | `(pressed: boolean) => void` | — | Callback when state changes |
| `disabled` | `boolean` | `false` | Disabled state |
| `className` | `string` | — | Additional CSS classes |

### ToggleGroup Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `type` | `'single' \| 'multiple'` | — | Selection mode |
| `value` | `string \| string[]` | — | Controlled value(s) |
| `onValueChange` | `(value: string \| string[]) => void` | — | Callback when value changes |
| `variant` | `'default' \| 'outline'` | `'default'` | Applied to all items |
| `size` | `'default' \| 'sm' \| 'lg'` | `'default'` | Applied to all items |
| `className` | `string` | — | Additional CSS classes |

### ToggleGroupItem Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | — | The value this item represents |
| `disabled` | `boolean` | `false` | Disabled state |
| `className` | `string` | — | Additional CSS classes (overrides group variant/size) |
