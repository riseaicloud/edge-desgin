# Input

A styled text input field that supports all native HTML input types.

## Import

```tsx
import { Input } from '@riseaicloud/ui'
```

## Usage

<Preview center={false} bgColor="#EFF4F9">
  <Input placeholder="搜索用户名" className="max-w-xs" />
</Preview>

```tsx
{/* 用户管理页面搜索框 */}
<Input placeholder="搜索用户名" />
```

## API

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `type` | `string` | `'text'` | HTML input type |
| `placeholder` | `string` | — | Placeholder text |
| `disabled` | `boolean` | `false` | Disabled state |
| `className` | `string` | — | Additional CSS classes |

All standard HTML `input` attributes are supported.
