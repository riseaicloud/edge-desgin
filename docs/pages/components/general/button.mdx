# Button

A button component with multiple variants and sizes, built on HTML `<button>` with support for links via `asChild`.

## Import

```tsx
import { Button, buttonVariants } from '@riseaicloud/ui'
```

## Usage

列表页标准工具栏按钮组合（来自用户管理、PVC、ConfigMap 等页面）。

<Preview bgColor="#EFF4F9">
  <Button variant="outline" size="sm" className="h-8 bg-blue-600 hover:bg-blue-700 text-white hover:text-white"><Plus className="mr-2 h-4 w-4" /> 新建用户</Button>
  <Button variant="outline" size="sm" className="h-8 w-8 p-0"><RotateCw className="h-4 w-4" /></Button>
  <Button variant="ghost" size="sm" className="h-6 w-6 p-0"><MoreHorizontal className="h-4 w-4" /></Button>
</Preview>

```tsx
import { Plus, RotateCw, MoreHorizontal } from 'lucide-react'

{/* 列表页主操作按钮 — 用户管理 / PVC / ConfigMap 等页面 */}
<Button variant="outline" size="sm" className="h-8 bg-blue-600 hover:bg-blue-700 text-white hover:text-white">
  <Plus className="mr-2 h-4 w-4" /> 新建用户
</Button>

{/* 图标刷新按钮 */}
<Button variant="outline" size="sm" className="h-8 w-8 p-0" onClick={refetch}>
  <RotateCw className="h-4 w-4" />
</Button>

{/* 表格行操作触发按钮 */}
<Button variant="ghost" size="sm" className="h-6 w-6 p-0">
  <MoreHorizontal className="h-4 w-4" />
</Button>
```

## As Link

Use the `asChild` prop to render as a different element:

```tsx
import Link from 'next/link'

<Button asChild>
  <Link href="/boss/users/new">新建用户</Link>
</Button>
```

## API

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'default' \| 'destructive' \| 'outline' \| 'secondary' \| 'ghost' \| 'link'` | `'default'` | Visual style variant |
| `size` | `'default' \| 'sm' \| 'lg' \| 'icon'` | `'default'` | Button size |
| `asChild` | `boolean` | `false` | Render as child element (uses Radix Slot) |
| `disabled` | `boolean` | `false` | Disabled state |
| `className` | `string` | — | Additional CSS classes |

All standard HTML button attributes are also supported.

### buttonVariants

Utility function for applying button styles to non-button elements:

```tsx
import { buttonVariants } from '@riseaicloud/ui'
import Link from 'next/link'

<Link href="/boss/users/new" className={buttonVariants({ variant: 'outline' })}>
  新建用户
</Link>
```
