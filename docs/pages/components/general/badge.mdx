# Badge

A small status label used to highlight information such as counts, tags, or states.

## Import

```tsx
import { Badge, badgeVariants } from '@riseaicloud/ui'
```

## As a link

Use `badgeVariants` to apply badge styles to non-badge elements:

```tsx
import Link from 'next/link'
import { badgeVariants } from '@riseaicloud/ui'

<Link href="/boss/users?status=pending" className={badgeVariants({ variant: 'outline' })}>
  待审核
</Link>
```

## API

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'default' \| 'secondary' \| 'destructive' \| 'outline'` | `'default'` | Visual style variant |
| `className` | `string` | — | Additional CSS classes |

All standard HTML `div` attributes are also supported.
