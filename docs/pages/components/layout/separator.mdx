# Separator

A visual divider between content sections, supporting horizontal and vertical orientations.

## Import

```tsx
import { Separator } from '@riseaicloud/ui'
```

## Usage

<Preview center={false}>
  <div className="w-64">
    <p className="text-sm">Above the separator</p>
    <Separator className="my-3" />
    <p className="text-sm">Below the separator</p>
  </div>
</Preview>

```tsx
<div>
  <p>Above the separator</p>
  <Separator />
  <p>Below the separator</p>
</div>
```

## API

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `orientation` | `'horizontal' \| 'vertical'` | `'horizontal'` | Direction of the separator |
| `decorative` | `boolean` | `true` | If true, hidden from accessibility tree |
| `className` | `string` | — | Additional CSS classes |

All standard HTML `div` attributes are also supported.
