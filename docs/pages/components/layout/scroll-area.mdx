# ScrollArea

A custom scrollable container with styled scrollbars, built on Radix UI ScrollArea.

## Import

```tsx
import { ScrollArea, ScrollBar } from '@riseaicloud/ui'
```

## Usage

<Preview>
  <ScrollArea className="h-48 w-48 rounded-md border">
    <div className="p-4">
      {['Namespace A', 'Namespace B', 'Namespace C', 'Namespace D', 'Namespace E', 'Namespace F', 'Namespace G', 'Namespace H'].map((item) => (
        <div key={item} className="py-2 text-sm border-b last:border-0">{item}</div>
      ))}
    </div>
  </ScrollArea>
</Preview>

```tsx
<ScrollArea className="h-72 w-48 rounded-md border">
  <div className="p-4">
    {items.map((item) => (
      <div key={item} className="py-2 text-sm">{item}</div>
    ))}
  </div>
</ScrollArea>
```

## Horizontal scrolling

<Preview center={false}>
  <ScrollArea className="w-80 whitespace-nowrap rounded-md border">
    <div className="flex gap-4 p-4">
      {['Node 1', 'Node 2', 'Node 3', 'Node 4', 'Node 5', 'Node 6', 'Node 7'].map((item) => (
        <div key={item} className="w-24 shrink-0 rounded bg-muted px-3 py-2 text-sm text-center">{item}</div>
      ))}
    </div>
    <ScrollBar orientation="horizontal" />
  </ScrollArea>
</Preview>

```tsx
<ScrollArea className="w-96 whitespace-nowrap rounded-md border">
  <div className="flex gap-4 p-4">
    {items.map((item) => (
      <div key={item} className="w-24 shrink-0">{item}</div>
    ))}
  </div>
  <ScrollBar orientation="horizontal" />
</ScrollArea>
```

## API

### ScrollArea Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | — | Additional CSS classes |
| `children` | `ReactNode` | — | Scrollable content |

All Radix UI ScrollArea Root props are supported.

### ScrollBar Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `orientation` | `'vertical' \| 'horizontal'` | `'vertical'` | Scrollbar direction |
| `className` | `string` | — | Additional CSS classes |
