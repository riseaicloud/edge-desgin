# Spacing

Edge Design System spacing tokens provide a consistent scale for padding, margin, gap, and sizing.

## Spacing Scale

| Token | Value |
|-------|-------|
| `0` | 0 |
| `px` | 1px |
| `0.5` | 0.125rem (2px) |
| `1` | 0.25rem (4px) |
| `1.5` | 0.375rem (6px) |
| `2` | 0.5rem (8px) |
| `2.5` | 0.625rem (10px) |
| `3` | 0.75rem (12px) |
| `4` | 1rem (16px) |
| `5` | 1.25rem (20px) |
| `6` | 1.5rem (24px) |
| `8` | 2rem (32px) |
| `10` | 2.5rem (40px) |
| `12` | 3rem (48px) |
| `16` | 4rem (64px) |
| `20` | 5rem (80px) |
| `24` | 6rem (96px) |

```tsx
<div className="p-4 gap-2 m-8">Content</div>
```

## Border Radius

| Token | Value |
|-------|-------|
| `rounded-none` | 0 |
| `rounded-sm` | calc(var(--radius) - 4px) |
| `rounded-md` | calc(var(--radius) - 2px) |
| `rounded-lg` | var(--radius) |
| `rounded-xl` | calc(var(--radius) + 4px) |
| `rounded-full` | 9999px |

The base `--radius` CSS variable defaults to `0.5rem` and can be overridden in your theme.

## Shadows

| Token | Value |
|-------|-------|
| `shadow-sm` | `0 1px 2px 0 rgb(0 0 0 / 0.05)` |
| `shadow-md` | `0 1px 3px 0 rgb(0 0 0 / 0.1), 0 1px 2px -1px rgb(0 0 0 / 0.1)` |
| `shadow-lg` | `0 4px 6px -1px rgb(0 0 0 / 0.1), 0 2px 4px -2px rgb(0 0 0 / 0.1)` |
| `shadow-xl` | `0 10px 15px -3px rgb(0 0 0 / 0.1), 0 4px 6px -4px rgb(0 0 0 / 0.1)` |

## Z-Index Scale

| Token | Value | Typical Usage |
|-------|-------|---------------|
| `z-[-1]` | -1 | Hide behind background |
| `z-0` | 0 | Default stacking |
| `z-[1000]` | 1000 | Dropdowns |
| `z-[1100]` | 1100 | Sticky elements |
| `z-[1200]` | 1200 | Overlays |
| `z-[1300]` | 1300 | Modals |
| `z-[1400]` | 1400 | Popovers |
| `z-[1500]` | 1500 | Toasts |
| `z-[1600]` | 1600 | Tooltips |

## Breakpoints

| Token | Value |
|-------|-------|
| `sm` | 640px |
| `md` | 768px |
| `lg` | 1024px |
| `xl` | 1280px |
| `2xl` | 1400px |

## Usage in TypeScript

```ts
import { spacing, radius, shadows, zIndex, breakpoints } from '@riseaicloud/tokens'
```
