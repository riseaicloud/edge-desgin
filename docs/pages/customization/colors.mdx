# Colors & Themes

Edge Design uses a two-tier color system:

1. **Semantic CSS custom properties** — HSL channel values that map to Tailwind utilities
2. **Theme presets** — Complete color configurations for header, sidebar, and navigation

## Semantic Colors

All semantic colors are defined as HSL channel values on `:root` and `.dark`, enabling Tailwind's opacity modifier syntax:

```css
:root {
  --primary: 221.2 83.2% 53.3%;
  --primary-foreground: 210 40% 98%;
}
```

```html
<div class="bg-primary text-primary-foreground">
  Solid primary
</div>
<div class="bg-primary/50">
  50% opacity primary
</div>
```

### Surface Scale

Page chrome uses a four-step surface scale. The order is the design language:
**page (lowest, darkest) < section < toolbar < card (raised, lightest)** — a gray page
wash with white cards floating on top. Dark mode mirrors the same relative order rather
than flattening everything to one value.

```tsx
// ✅ Follows the theme in both light and dark
<div className="bg-surface-page">
  <div className="bg-card border rounded">
    <div className="bg-surface-toolbar p-4">…</div>
  </div>
</div>

// ❌ Frozen — ignores dark mode and theme switches
<div style={{ backgroundColor: '#EFF4F9' }}>
```

<Callout type="warning">
  The `surfaceColors` **hex** export (`surfaceColors.page` etc.) is **deprecated**. It can
  only be used in inline styles, which never follow the theme. Use the `bg-surface-*`
  utilities instead. The hex export stays for backward compatibility until 2.0.
</Callout>

### Using Themes

```tsx
import { themes, getTheme } from '@riseaicloud/tokens'

// Get a theme
const theme = getTheme('purple')

// Apply to header
<header className={theme.colors.headerBg}>
  <span className={theme.colors.text}>Edge Platform</span>
</header>
```

### Custom Themes

```tsx
import { registerTheme } from '@riseaicloud/tokens'
import type { Theme } from '@riseaicloud/tokens'

const myTheme: Theme = {
  name: 'custom',
  label: 'My Custom Theme',
  colors: {
    headerBg: 'bg-[#1a1a2e]',
    headerBorder: 'border-indigo-900',
    // ... all ThemeColors fields
  },
}

registerTheme(myTheme)
```

## Overriding Tokens

Override any CSS custom property in your global styles:

```css
:root {
  --primary: 262 83% 58%;    /* Change primary to purple */
  --radius: 0.75rem;          /* Larger border radius */
}
```

Or extend the Tailwind preset:

```js
const edgePreset = require('@riseaicloud/tokens/tailwind-preset')

module.exports = {
  presets: [edgePreset],
  theme: {
    extend: {
      colors: {
        brand: {
          DEFAULT: 'hsl(262 83% 58%)',
          foreground: 'hsl(0 0% 100%)',
        },
      },
    },
  },
}
```
