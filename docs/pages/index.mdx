# Edge Design

A comprehensive design system for the Edge Platform — providing a consistent design language, reusable components, and composable patterns for building cloud-edge management interfaces.

## Features

- **Design Tokens** — Colors, typography, spacing, and motion tokens in CSS variables, Tailwind preset, and JS constants
- **6 Built-in Themes** — Classic Blue, Purple, Green, Orange, Dark, and Cyan with custom theme extensibility
- **50+ Components** — Core UI components built on Radix UI with full accessibility support
- **Domain Components** — Edge-specific selectors, monitoring dashboards, topology visualization
- **Hooks Library** — Reusable React hooks for state management, UI interactions, and utilities
- **Pluggable Architecture** — Token-first design enables complete rebranding; layered packages allow selective adoption

## Packages

| Package | Description |
|---------|-------------|
| `@riseaicloud/tokens` | Design tokens — CSS variables, Tailwind preset, JS/TS constants |
| `@riseaicloud/ui` | Core UI components — Button, Input, Dialog, Table, Card, etc. |
| `@riseaicloud/edge-components` | Domain components — Selectors, Monitoring, Topology, Terminal |
| `@riseaicloud/hooks` | Shared hooks — State management, UI, DOM utilities |
| `@riseaicloud/icons` | Icon system — Lucide React + custom Edge icons |

## Quick Start

### Install

```bash
pnpm add @riseaicloud/ui @riseaicloud/tokens
```

### Use the Tailwind Preset

```js
// tailwind.config.js
const edgePreset = require('@riseaicloud/tokens/tailwind-preset')

module.exports = {
  presets: [edgePreset],
  content: ['./src/**/*.{ts,tsx}'],
}
```

### Use Components

```tsx
import { Button, Card, Badge } from '@riseaicloud/ui'

function App() {
  return (
    <Card>
      <Badge variant="success">Running</Badge>
      <Button>Deploy</Button>
    </Card>
  )
}
```
