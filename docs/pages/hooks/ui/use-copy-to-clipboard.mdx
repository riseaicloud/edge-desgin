# useCopyToClipboard

Copies text to the clipboard using the Clipboard API. Provides a `copied` state that auto-resets after a delay.

## Import

```tsx
import { useCopyToClipboard } from '@riseaicloud/hooks'
```

## Usage

```tsx
const { copied, copy } = useCopyToClipboard()

return (
  <button onClick={() => copy('https://edge.example.com')}>
    {copied ? 'Copied!' : 'Copy link'}
  </button>
)
```

## Custom reset delay

```tsx
// Reset "copied" state after 5 seconds instead of default 2s
const { copied, copy } = useCopyToClipboard(5000)
```

## With token display

```tsx
const { copied, copy } = useCopyToClipboard()

return (
  <div className="flex items-center gap-2">
    <code>{token}</code>
    <button onClick={() => copy(token)}>
      {copied ? <CheckIcon /> : <CopyIcon />}
    </button>
  </div>
)
```

## API

### Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `resetDelay` | `number` | `2000` | Milliseconds before `copied` resets to `false` |

### Returns

| Value | Type | Description |
|-------|------|-------------|
| `copied` | `boolean` | `true` for `resetDelay` ms after a successful copy |
| `copy` | `(text: string) => Promise<boolean>` | Copy text; resolves `true` on success |

### Notes

- Returns `false` if `navigator.clipboard` is unavailable (non-HTTPS, unsupported browser)
- `copied` automatically resets after `resetDelay`
