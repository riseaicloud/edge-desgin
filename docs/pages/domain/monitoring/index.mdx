# Monitoring

Components for displaying monitoring data, metrics, events, and alerts.

## Layer Overview

| Layer | Package | Components |
|-------|---------|------------|
| **Layer 2** | `@riseaicloud/ui` | `KPICard`, `ResourceChart` |
| **Layer 3** | `@riseaicloud/components` | `MetricCard`, `NamespaceStatsCards`, `NodeStatsCards`, `EventsTable`, `EventsViewer`, `AlertList`, `AlertListContainer` |

## Layer Rules

- **Layer 2** components are generic — no K8s domain knowledge, no API calls.
- **Layer 3** components know about Edge Platform concepts (namespaces, nodes, alerts) but still do **not** call APIs directly.
