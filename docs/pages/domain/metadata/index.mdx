# Metadata

Kubernetes labels and annotations display components. These render `metadata.labels` and `metadata.annotations` from any K8s resource in a consistent, readable format.

## Import

```tsx
import { LabelsDisplay, AnnotationsDisplay } from '@riseaicloud/components'
```

---

Renders Kubernetes labels as inline badge pairs. Each label is displayed as a split pill: dark background for the key, light background for the value.

```tsx
<AnnotationsDisplay
  annotations={resource.metadata.annotations}
/>
```

### Custom Column Layout

By default the key column takes 3/12 and the value column takes 9/12. Adjust with `colSpan`:

```tsx
<AnnotationsDisplay
  annotations={resource.metadata.annotations}
  colSpan={[4, 8]}
/>
```

## Usage Pattern

Both components are typically used inside a detail panel or `Descriptions` block:

```tsx
import { LabelsDisplay, AnnotationsDisplay } from '@riseaicloud/components'

function ResourceMetadataTab({ resource }: { resource: KubernetesResource }) {
  return (
    <div className="space-y-6">
      <section>
        <h3 className="text-sm font-medium text-gray-700 mb-2">ć ‡ç­ľ</h3>
        <LabelsDisplay labels={resource.metadata.labels} />
      </section>

      <section>
        <h3 className="text-sm font-medium text-gray-700 mb-2">ćł¨č§Ł</h3>
        <AnnotationsDisplay annotations={resource.metadata.annotations} />
      </section>
    </div>
  )
}
```
