# Resource Display

Status and resource metric display components for Edge Platform. These components present cluster, node, and resource data in a consistent visual style.

## Import

```tsx
import {
  ClusterInfoCard,
  NodeCard,
  StatCard, MiniStat, NetworkStat, PodStat,
  RingGauge, RingGaugeGroup,
  ResourceMetricCard,
} from '@riseaicloud/components'
```

| `showEdgeXButton` | `boolean` | — | Force show EdgeX button |

| `className` | `string` | `''` | Additional CSS classes |

**Status thresholds:** `warning` when any resource ≥ 80%, `offline` when `isReady` is `false`.

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `number` | — | Percentage 0–100 |
| `label` | `string` | — | Label text below gauge |
| `unit` | `string` | `'%'` | Unit for display |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Gauge size |
| `colorMode` | `RingGaugeColorMode` | `'auto'` | Color mode (`auto` uses threshold coloring) |
| `animated` | `boolean` | `true` | Animate value changes |
| `displayValue` | `string` | — | Override percentage display |
| `strokeWidth` | `number` | — | Ring stroke width override |
| `className` | `string` | `''` | Additional CSS classes |

**Auto-color thresholds:** < 70% → cyan, 70–85% → yellow, ≥ 85% → red.

## ResourceMetricCard

Compact resource selection card with progress bar. Used for switching between resource views (CPU / Memory / Disk).

<Preview bgColor="#EFF4F9">
  <div className="flex gap-3">
    <ResourceMetricCard
      type="cpu"
      currentValue={4.2}
      totalValue={8.0}
      unit=" cores"
      selected={true}
      onClick={() => {}}
    />
    <ResourceMetricCard
      type="memory"
      currentValue={11.4}
      totalValue={16.0}
      unit=" GB"
      selected={false}
      onClick={() => {}}
    />
    <ResourceMetricCard
      type="disk"
      currentValue={234}
      totalValue={500}
      unit=" GB"
      selected={false}
      onClick={() => {}}
    />
  </div>
</Preview>

```tsx
const [activeMetric, setActiveMetric] = useState<ResourceMetricType>('cpu')

{(['cpu', 'memory', 'disk'] as ResourceMetricType[]).map(type => (
  <ResourceMetricCard
    key={type}
    type={type}
    currentValue={metrics[type].current}
    totalValue={metrics[type].total}
    unit={metrics[type].unit}
    selected={activeMetric === type}
    onClick={() => setActiveMetric(type)}
  />
))}
```

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `type` | `'cpu' \| 'memory' \| 'disk'` | — | Resource type (controls icon and color) |
| `currentValue` | `number` | — | Current usage |
| `totalValue` | `number` | — | Total capacity |
| `unit` | `string` | — | Unit string (e.g. `" cores"`, `" GB"`) |
| `selected` | `boolean` | — | Highlight with blue border |
| `onClick` | `() => void` | — | Selection callback |
| `className` | `string` | `''` | Additional CSS classes |

**Color scheme:** CPU → blue, Memory → green, Disk → orange.
